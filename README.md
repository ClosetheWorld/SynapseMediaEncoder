# SynapseMediaEncoder

## Hapエンコード
VJソフト SynapseRack向けのHapエンコードソフト  
FFmpegのGUIラッパーです。

## 使い方
1.SynapseMediaEncoder.exeを起動  
2."Select"からffmpeg.exeのパスを指定（指定されていない場合）
3.エンコードしたい動画ファイルをD&D  
4.エンコード形式と解像度を指定  
5."Encode"をクリック
